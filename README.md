# 📊 Meta Ad Performance Dashboard (Power BI)

## 🚀 Overview
This project is an interactive **Power BI dashboard** designed to analyze **Meta (Facebook & Instagram) ad performance**.  
It converts raw marketing data into meaningful insights to support **data-driven decision making**.

---

## 🎯 Objectives
- Track and monitor key ad performance metrics  
- Understand audience behavior and engagement  
- Compare different ad formats  
- Identify trends for campaign optimization  

---

## 📌 Key Metrics
- Impressions: 112.4K  
- Clicks: 12.9K  
- Engagements: 14.9K  
- CTR: 11.48%  
- Conversion Rate: 5.25%  
- Purchase Rate: 0.60%  

---

## 📊 Features
- KPI cards for quick insights  
- Audience segmentation (Age, Gender)  
- Geographic analysis (Clicks by Country)  
- Time-based trends (Weekly & Hourly)  
- Ad type performance comparison  
- Interactive filters and slicers  
- Dynamic dashboard navigation  

---

## 🛠️ Tools & Technologies
- Power BI  
- DAX (Data Analysis Expressions)  
- Data Modeling  
- Data Visualization  

---

## 💡 Key Insights
- Identified top-performing campaigns and ad formats  
- Found peak engagement times for better scheduling  
- Improved targeting through audience segmentation  
- Highlighted low-performing areas for optimization
